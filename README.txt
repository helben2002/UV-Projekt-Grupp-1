Bygg din egen simple banking system(SBS):
Gå via terminalen till katalogen där du vill att ditt SBS ska ligga
Skriv in: 
1. 'git clone https://github.com/helben2002/UV-Projekt-Grupp-1.git'
2. 'cd UV-Projekt-Grupp-1.git'
3. 'mvn compile'
4. 'mvn test'
5. 'mvn package'
6. 'java -cp target/UV-Projekt-Grupp-1-1.0-SNAPSHOT.jar se.yrgo.Main'

Nu kan du testa ditt egna Simple Banking System. Lycka till!
